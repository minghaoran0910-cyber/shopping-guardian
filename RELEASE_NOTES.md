# 购物守护者 v1.1.0

Android 首个可安装版本。现在可以直接从淘宝、天猫或京东的系统分享面板导入链接，也可以在设备上识别购物车截图。

## Android 新增

- 接收系统分享的商品文字和链接
- 支持应用冷启动和已打开状态下的新分享
- 系统文件选择器，不读取整个相册
- 随 APK 提供的 ML Kit 中文 OCR，识别不依赖用户模型
- Android Keystore 支持的 API Key 安全存储
- 冷静期本地通知，支持创建和取消
- Android 13 及以上按需请求通知权限

## 现有功能

- 淘宝、天猫单商品分享链接
- 京东单商品和购物清单分享链接
- 淘宝、天猫、京东购物车截图，本地 OCR
- 手动商品录入和导入结果编辑
- JustOneAPI 商品详情补全
- OpenAI-compatible 模型配置、连接测试和结构化建议
- 月度非必要消费预算
- 可新增、启停和删除的个人消费规则
- 购买、等待、放弃、寻找替代四种决定
- 冷静期页面和本地通知
- 本地历史搜索、筛选、详情、反馈和删除
- 基于真实记录的基础习惯统计
- 不含 API Key 的 JSON 导出
- 本地业务数据与 API 配置独立清除

## 已知限制

- 淘宝购物车分享链接可能只允许淘宝 App 打开，建议使用购物车截图。
- OCR 结果受截图清晰度和页面排版影响，分析前请核对。
- iOS 安装包尚未发布。
- Android 系统分享目前只接收文字和链接，截图需在应用内选择。
