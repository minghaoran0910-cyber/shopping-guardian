# 购物守护者 v1.0.0

第一个可用的 macOS 版本。它已经可以完成一次完整流程：导入商品，核对信息，结合预算与个人规则调用自己的模型，记录决定，等待冷静期，并在之后补充是否后悔。

## 主要功能

- 淘宝、天猫单商品分享链接
- 京东单商品和购物清单分享链接
- 淘宝、天猫、京东购物车截图，本地 Vision OCR
- 手动商品录入和导入结果编辑
- JustOneAPI 商品详情补全
- OpenAI-compatible 模型配置、连接测试和结构化建议
- 月度非必要消费预算
- 可新增、启停和删除的个人消费规则
- 购买、等待、放弃、寻找替代四种决定
- 冷静期页面和 macOS 本地通知
- 本地历史搜索、筛选、详情、反馈和删除
- 基于真实记录的基础习惯统计
- 不含 API Key 的 JSON 导出
- 本地业务数据与 API 配置独立清除

## 安装提示

安装包尚未使用 Apple Developer ID 签名和公证。首次打开时，请在 Finder 中右键“购物守护者”，选择“打开”，再确认一次。

## 已知限制

- 淘宝购物车分享链接可能只允许淘宝 App 打开，建议使用购物车截图。
- OCR 结果受截图清晰度和页面排版影响，分析前请核对。
- Android 和 iOS 安装包尚未发布。
