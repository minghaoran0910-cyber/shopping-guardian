# 购物守护者 v0.1.0-alpha.1

这是第一个可下载的 macOS 测试版，重点先把商品导入跑通。

## 这版可以做什么

- 解析京东购物清单和京东单商品分享链接
- 解析淘宝、天猫单商品分享链接
- 选择淘宝、天猫、京东购物车截图并在本地识别
- 使用 JustOneAPI 补充商品标题、价格和图片
- 在导入前预览商品
- 保存主题、语言和 JustOneAPI 设置

## 使用前需要知道

- 当前安装包没有 Apple Developer ID 签名和公证。首次启动请在 Finder 中右键应用并选择“打开”。
- 淘宝购物车分享链接可能受登录和风控限制，推荐使用截图导入。
- 截图识别后请核对商品标题和价格。
- 消费分析、本地历史记录、Android 和 iOS 安装包仍在开发中。

详细步骤见 [README](README.md)。
